# 祥瑞白板录制工具操作说明

祥瑞白板录制工具用于录制白板讲解、课程演示和异步说明视频。

## 快速开始

1. 在白板上绘制内容，或从素材库导入常用组件。
2. 点击右上角录制按钮，选择录制比例和画面范围。
3. 在录制设置中调整背景、圆角、画布边距、摄像头和音频。
4. 点击开始录制，完成后导出视频。

## 录制设置

- 录制：选择 16:9、4:3、3:4、9:16、1:1 或自定义尺寸。
- 背景：选择内置背景，或上传图片并适配录制比例。
- 摄像头：选择视频源，切换圆形或方形，调整摄像头大小。
- 音频：选择麦克风源，控制麦克风和声音开关。
- 光标：控制录制时的鼠标高亮效果。

## 素材库

点击右上角素材库按钮可以打开侧边素材库。公共素材库支持搜索、刷新、添加到白板和下载。

## 快捷键

左上角菜单中可以打开快捷键设置，自定义录制、面板和编辑器偏好相关快捷键。

## 开源致谢

白板编辑能力基于 Excalidraw，录制和媒体处理使用多个开源项目。第三方来源、许可证和修改记录会在项目文档中持续维护。
