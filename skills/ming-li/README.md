# ming-li

> 一个专业命理分析的 Claude Code Skill。
> 八字四家合一 + 紫微斗数 + 六爻文王课，骨架来自韦千里 8 步范式 + ISAR 国际占星标准 + 心理咨询 SOAP 流程。
> 输出新中式古典风的 HTML 长卷 + 2× DPI PNG 长图，方便发朋友圈或保留在 vault。

---

## ✦ 这个 skill 解决什么问题

市面上"DeepSeek/GPT 算命" prompt 模板满天飞，但大部分都是把命理学当玩梗：

- 让 AI 直接扮演"精通《滴天髓》《三命通会》的命理大师"，几秒钟一份报告
- 报告里堆 PMF / Cash Cow / GTD / INTJ / HSP 一堆**英文术语**包装
- 出来的东西看上去专业，**但既不是命理学，也不是心理学**

`ming-li` 走另一条路：

1. **把命理学当一门"专业知识"对待**——按韦千里 1935 年定下的《千里命稿》8 步范式（看强弱 / 定格局 / 取用神 / 论喜忌 / 查岁运 / 推六亲 / 评性情 / 断事业）走流程
2. **四家合一互证**——陆致极做骨架、任铁樵补深度、沈孝瞻定格局、万民英补神煞，三家以上一致才下判语
3. **借西医专业化路径**——ISAR 国际占星协会 + 心理咨询 SOAP 流程做参照
4. **13 类全民赛道**——传统八类（仕途 / 文教 / 医卫 / 司法军警 / 商贾 / 工艺 / 农林 / 服务）+ 现代五类（文创 / 自由职业 / 创业投资 / 科研 / 文娱体育），覆盖中国社会真实人群
5. **三层通俗化**——古今夹叙正文 + 行内浅注 + 卷末术语简注 + 每章一句话总结，让 60 岁退休工人和 25 岁互联网青年都能读懂
6. **严守 12 条伦理红线**——不预测寿命 / 不算婚配 / 不替代医疗法律心理 / 不诱导付费化解

## ✦ 让 Agent 自动安装（推荐）

把下面这句话贴给你的 AI Agent（Claude Code / Cursor / aider 都行）：

> **「帮我安装 https://github.com/Larkin0302/vantasma-toolkit 里的 ming-li skill（路径 `skills/ming-li/`）。把整个目录 clone / 复制到 `~/.claude/skills/ming-li/`，然后给我一句话演示触发：『帮我算个八字，1990 年 5 月 1 日上午 10 点，山西出生』。」**

Agent 会自动跑完。需要你介入的只有：

- 一次 `gh auth login`（如果没登 GitHub）
- 装一下 `Pillow`（PNG 导出依赖）：`pip3 install Pillow`
- 装一个 Chrome 或 Chromium（HTML → PNG 截图用 headless 模式）

## ✦ 手动安装

```bash
# 1. 克隆仓库
git clone https://github.com/Larkin0302/vantasma-toolkit.git
cd vantasma-toolkit

# 2. 复制 skill 到 Claude Code skills 目录
cp -r skills/ming-li ~/.claude/skills/

# 3. 装 PNG 导出依赖
pip3 install Pillow --break-system-packages

# 4. 确保 Chrome 已装（HTML → PNG 用）
#    macOS: /Applications/Google Chrome.app
#    Linux: apt install chromium 或 brew install --cask google-chrome

# 5. 重启 Claude Code，跟 Agent 说："帮我算个八字"
```

## ✦ 怎么用

在 Claude Code 里说：

> 帮我算个八字，1990 年 5 月 1 日上午 10 点 15 分，山西出生

或：

> 帮我看一下紫微命盘，[出生时间]

或：

> 摇一卦问 [事情]，结果是 [卦象]

skill 自动触发，按 8 步流程跑：

| Step | 干什么 |
|---|---|
| 1 | 真太阳时换算 + 排八字四柱（含大运逆顺） |
| 2 | 看强弱（日主旺衰 + 五行配比） |
| 3 | 定格局（建禄 / 月劫 / 食伤 / 财官 / 印重 / 七杀等八格 + 变格） |
| 4 | 取用神 + 论喜忌 |
| 5 | 查岁运（大运 10 年一柱 + 流年） |
| 6 | 推六亲（父母 / 配偶 / 子女 / 兄弟） |
| 7 | 评性情 + 13 类全民赛道适配 |
| 8 | 断事业 + 一句话总诀 |

输出 HTML 卷轴（新中式古典风：宣纸米黄 + 楷宋 + 朱砂印章 + 四角角花），可选导出 PNG 长图（2× DPI Retina 级清晰，便于发朋友圈）。

## ✦ 不做什么（严守红线）

- ❌ 不预测寿命 / 死期
- ❌ 不算婚配（不评价两个人合不合）
- ❌ 不替代医疗、法律、心理咨询
- ❌ 不诱导付费化解（不卖法器 / 不接化煞订单）
- ❌ 不算别人（必须命主本人在场）
- ❌ 不评价他人福薄
- ❌ 不强化宿命论（一切判语都附"可调整方向"）

完整 12 条见 `references/04-伦理代码.md`。
