# 八字推演（陆致极 8 步法 · 四家合判）

> 八字推演主流程。陆致极方法做骨架，任铁樵 / 沈孝瞻 / 万民英补深度。
> 四家详细方法论见 `14-八字-四家对照.md`，本文是**操作步骤**。

## 步骤 1：收集生辰

**必须确认**：
- 公历 / 农历（哪个）+ 年月日
- 出生时刻（精确到分）
- 出生地（真太阳时校正）
- 性别（决定大运顺逆）

**若没给齐**：主动追问，**不要瞎猜**。

## 步骤 2：排四柱

按 `01-排盘基础.md`：
1. **年柱**：立春为界（不是春节）
2. **月柱**：节气为界（立春、惊蛰、清明、立夏...）
3. **日柱**：万年历查
4. **时柱**：五鼠遁

## 步骤 3：地支藏遁展开

查 `01-排盘基础.md` § 5 地支藏遁表。

## 步骤 4：身强身弱（陆致极三步法）

### 第 1 步：季节权重

| 月支 | 旺 |
|---|---|
| 寅卯辰 | 木旺 |
| 巳午未 | 火旺 |
| 申酉戌 | 金旺 |
| 亥子丑 | 水旺 |
| 辰戌丑未 | 土旺 |

日主五行**得令** → 基础值 +2

### 第 2 步：干支生克累计

遍历四柱（除日干），每个干支对日主：
- 生 → +1
- 克 → −1
- 泄 → −0.5
- 同类 → +0.5

### 第 3 步：月令绝对加权

- 月支本气 = 日主五行 → **+2**
- 月支中气 = 日主五行 → +1.5
- 月支余气 = 日主五行 → +0.8

**结论**：综合分 ≥ 0.5 → 身强；< 0.5 → 身弱

## 步骤 4.5：气势论补判（任铁樵）

陆致极算完后用任铁樵检验：
- **整体气聚不聚**？四柱是否形成"一股力"？
- 用神**是否有根**（透干或地支本气）？
- 是否**旺极反凶**（极旺需运中遇克才平衡）？

## 步骤 5：取用神

### 陆致极简易法
- **身强**：候选 = 官杀 > 食伤 > 财
- **身弱**：候选 = 印 > 比肩 > 劫

### 任铁樵补深度
- 用神**必须能克制病神**才有效
- 用神无根 → 整个格局失效

### 格局评分（满分 10）
- 用神到位得气 +3
- 用神无破损 +2
- 用神数量适中（1-2）+2
- 忌神距日主远 +2
- 大运配合 +1

→ ≥ 7：清格 / 5-6：中平 / < 5：浑浊

## 步骤 5.5：格局判定（沈孝瞻法）

**核心原则**：八字用神，专求月令

1. 看月支本气什么十神 → 落入八格
2. 月支为日干禄/劫 → **建禄格 / 月劫格**（特殊处理）
3. 用神必须透干 → 用神无透 = 格局未成 = 待大运补
4. **行运成格**：原局未成 → 大运补缺 → 格局成立

详见 `13-八字-格局学.md`。

## 步骤 5.6：神煞补查（万民英）

详见 `11-八字-神煞库.md`。

**重点查**：
- 天乙贵人 / 驿马 / 桃花 / 华盖 / 红艳
- **截路空亡** / 阴阳差错 / 十恶大败 / 三奇贵神
- 国印 / 福星 / 将星 / 攀鞍 / 学堂

## 步骤 6：排大运（关键 · 容易错）

### 顺逆规则（陆致极标准）

| 性别 | 年干阴阳 | 顺逆 |
|---|---|---|
| 男 | 阳干（甲丙戊庚壬）| **顺** |
| 男 | 阴干（乙丁己辛癸）| **逆** |
| 女 | 阳干 | **逆** |
| 女 | 阴干 | **顺** |

⚠️ **不要用"奇月顺/偶月逆"的规则**——那是错的。

### 起运年数

- 阳男阴女 → 数距下一个节气的天数
- 阴男阳女 → 数距上一个节气的天数
- 天数 ÷ 3 = 起运年数（1 天 = 4 个月）

### 大运排列

- **顺**：月柱干支 → 下一柱（甲寅→乙卯→丙辰）
- **逆**：月柱干支 → 上一柱（甲寅→癸丑→壬子）
- 每柱 10 年

### 沈孝瞻取运专表

每格有专属取运法，详见 `13-八字-格局学.md` § 各格取运专表。

⚠️ **沈孝瞻最重要的一条**：禄劫用伤食，**行官非吉，透官不美**——这跟陆致极版判断常有冲突。

## 步骤 7：流年推演

详见 `90-应期算法.md`。

**核心规则**：
- 流年干生日主 → 较吉
- 流年干克日主 → 较凶
- 流年支冲月令 → 大变化
- 流年支合用神 → 较吉
- 流年支冲日支 → 人际/感情波动

**最凶组合**：流年干克日主 + 流年支冲月令
**最吉组合**：流年支合用神 + 流年干生日主

## 重要纪律

### ✅ 该做的

1. **必用真太阳时**校正
2. **大运顺逆**严格按"阳男阴女顺/阴男阳女逆"
3. **格局判断**先看身强身弱，再取用神
4. **四家互证**——三家以上一致才高确信
5. **建议交叉验证**：推荐用户用工具校核

### ❌ 不该做的

1. **不替用户做"信不信"的价值判断**
2. **不算姻缘配对**（user.md 感情禁区）
3. **不预测寿命 / 生死 / 重大灾祸**
4. **不替代专业命理师**
5. **不用迷信化语言**："必定" → "命理倾向 / 通常意味着"
