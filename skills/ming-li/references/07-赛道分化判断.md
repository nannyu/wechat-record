# 赛道分化判断（13 类全民赛道 · 同一命局不同赛道不同解读）

> **v2.3 重大升级**：从原 9 类"互联网思维"赛道，扩展为 **13 类全民赛道**。
> 覆盖中国社会真实人群（公务员 / 医生 / 教师 / 农民 / 工人 / 商人 / 手艺人 / 家庭主妇 ...），
> 不再以"想做 IP / 想创业 / 想做学术"为默认。
>
> **基础**：来自八字传统职业适配体系（《千里命稿》《滴天髓》《三命通会》）+ 全网命理师共识。

## 一、为什么必须分赛道判

### 反例：粗暴用"大器晚成"

命局：**印重身旺 + 用神埋没 + 39 岁后大运启动**

不分赛道的粗暴判语：
> "你是大器晚成的命，39 岁前都在积累，39 岁后才真正启动。"

实际上同样这个命局：
- 在 **公务员**赛道：30-50 岁是关键期（升迁有年限，命格可正常副处副科）
- 在 **教师**赛道：教龄复利极合命格，越老越值钱
- 在 **农民**赛道：印重 ≠ 不勤劳，命格更耐得住季节性等待
- 在 **手艺人**赛道：印重善学技术，10 年成精
- 在 **互联网创业**赛道：39 岁才启动 = 错失窗口期，要调打法
- 在 **演艺**赛道："大器晚成"是误判，需提前出来

→ 同一句话，不同人群下含义完全不同。命理师不分赛道，就是不专业。

## 五、综合判 SOP（推命时使用）

### Step 1 · 先看格局总轮廓
按 `13-八字-格局学.md` 定格局。

### Step 2 · 问来访者**实际从事 / 意向赛道**

不是"想"什么，是"做什么 / 想做什么"。13 类速问：
> "你现在做的，或想做的，更接近以下哪一类？
> 1. 公务员 / 体制
> 2. 教师 / 学者 / 文化
> 3. 医生 / 护士 / 医药
> 4. 律师 / 警察 / 军人
> 5. 商人 / 金融 / 会计
> 6. 工程师 / 设计师 / 技工
> 7. 农户 / 实业 / 制造
> 8. 餐饮 / 家政 / 销售
> 9. 艺术 / 设计 / 内容
> 10. 自由职业 / 个人 IP
> 11. 创业 / 投资
> 12. 科研 / 科技
> 13. 演艺 / 体育"

### Step 3 · 查矩阵
本文 § 三 五维 × 13 赛道，§ 四 典型组合 × 赛道。

### Step 4 · 输出**朴素双语判语**

按 `06-古今夹叙表达.md` 的语言规范：
- 浅近文言 + 朴素白话
- **不堆英文**
- **不用破折号**
- 古典印证一段最多一句

例（示例命主 + 公职）：
> 命主之造，建禄印重，孤官无辅，又时支寅申冲身。
> 公职之路，须循资按规，重在持守与协作。然命主之骨格，独立自主之性极盛，难安一处。
> 古人云"良禽择木而栖"，木不对，再好的鸟也难展翅。
> 故公职非命主所宜，不建议入。

### Step 5 · 给出**赛道决策建议**

3 种情境：
- A. 来访者赛道选对了 → 恭喜并给具体行动
- B. 来访者赛道选错了 → 给**赛道切换建议**
- C. 来访者赛道一半对一半错 → 给**复合赛道方案**（主业 + 副业）

## 六、赛道决策建议库（按典型场景）

### 印重身旺者 · 想做公务员
> 不建议。命主孤官无辅，公职之路艰涩。若家中安排已入，则宜寻**技术官**而非**事务官**——如档案、文教、科研等内设岗位，离规则远些，离专业近些。

### 印重身旺者 · 已在工厂
> 适合做**技术工 / 工艺师**，不适合做工长 / 工头。命格"工匠路线"是命主之福，无须强求管理岗。古人讲"技近乎道"，做一行精一行，命格自然显贵。

### 印重身旺者 · 想做教师
> 极配。印重为学者标配，火木伤官（若有）尤宜文教。建议方向：**学科教学 + 业余写作 + 长期教研**。命格 40 岁后逐渐显贵，教龄复利极合。

### 印重身旺者 · 已是医生
> 看路：**中医 / 心理咨询 / 内科**合命格（印重善学，沉得住）；**外科 / 急诊**不合（用神不在杀气）。若已在外科，建议**转研究 / 教学方向**，做主任而非主刀。

### 印重身旺者 · 在做销售
> 不合。命格"慢启动 + 不善社交"，与销售"快反馈 + 重客情"冲突。建议**转顾问式销售**（如医药代表 / 行业咨询）或**转售前 / 产品方向**。

### 食伤旺者 · 已在体制
> 命格给的是"笔杆子 / 喉舌"，体制给的是"螺丝钉"。短期能熬，长期会窒息。若不能离开，宜寻**宣传 / 文化 / 教研**类岗，远离纯事务岗。

### 比劫旺者 · 想合伙创业
> 命格"一人成军"，合伙是逆结构。真要合作，**业务合伙不要股权合伙**，保持决策独立。古人云"宁为鸡口，无为牛后"。

### 财官双美者 · 在做自由职业
> 命格给的是"双轨道"（财 + 官），自由职业只用了一半。长期看会进入企业或机构。古人云"学而优则仕"——不一定做官，但需要一个组织承接命主之能。

## 七、赛道判断的伦理（同 v2.2）

### 不能做的事

1. ❌ **不替来访者选赛道**——只给信息，决策由来访者自定
2. ❌ **不否定来访者当前赛道**——给"调整方案"而非"否定 + 推倒"
3. ❌ **不刻板化**：命局只是倾向，努力 + 资源仍可改变结果
4. ❌ **不替职业咨询师 / 心理咨询师做事**

### 该做的事

1. ✅ 把**赛道选择的信息**摊给来访者
2. ✅ 提供**多种可能性**
3. ✅ 给**赛道内的优化建议**（不一定要换赛道）
4. ✅ 强调**自我认识 + 主动选择**

## 九、最终自检
