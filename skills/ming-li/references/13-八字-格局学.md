# 八字格局学（沈孝瞻八格 + 三命通会变格 + 普通正格判别）

> 合并：ba-zi/03-特殊格局.md + ba-zi/04-八格格局学.md + 三命通会内化补遗
> 格局学 = **沈孝瞻为骨架，万民英补变格**。

## 一、判格三层流程

### 第 1 层：是否为变格？

**约 15-20% 命局**为变格（从格 / 化气 / 专旺等），优先判别。变格成立则身强身弱判断**完全翻转**。

### 第 2 层：判八正格

月支本气是什么十神 → 落入八格之一：
- 正官 / 七煞 / 正财 / 偏财 / 食神 / 伤官 / 正印 / 偏印

### 第 3 层：建禄 / 月劫格

月支为日干禄/劫 → 不能用比劫当用神 → **看天干透出取财官煞食为用**

## 三、各格取运专表（沈孝瞻原句精华）

### 正官格
| 用法 | 取运 |
|---|---|
| 用财印 | 喜印绶身旺，**忌食伤** |
| 佩印 | 喜财乡，伤食反吉 |
| 带伤兼印 | 喜官旺印旺，财乡切忌 |

### 七煞格
> **平衡论**：哪边轻补哪边
| 用法 | 取运 |
|---|---|
| 食制煞 | **食重煞轻则助煞，食轻煞重则助食** |
| 用印 | 不利财乡，伤食为美 |
| 带财存食 | 喜身旺印绶 |

### 正/偏财格
| 用法 | 取运 |
|---|---|
| 财旺生官 | 不利七煞伤官 |
| 食伤生财 | 财重身轻喜助身，财轻身重宜助财 |
| 佩印 | **喜官乡**，身弱逢之最喜印旺 |
| 带七煞 | 喜食伤身旺之方 |

### 印绶格
| 用法 | 取运 |
|---|---|
| 用官 | 官露印重，财运反吉 |
| 带伤食 | 财运反吉 |
| 印用七煞 | 喜伤食身旺，**忌财乡** |
| 印多用财 | 喜财乡，逢印必凶 |

### 食神格
| 用法 | 取运 |
|---|---|
| 食神生财 | 财重食轻则行财食，财食重则喜帮身 |
| 食神带煞印 | 喜印旺，切忌财乡 |

### 伤官格
| 用法 | 取运 |
|---|---|
| 用财 | 财旺身轻则印比，身强财浅则财运伤食运亦宜 |
| 佩印 | 运行官煞为宜，印地亦吉 |
| **见官** | ⚠️ **伤官见官，祸患百端**——除非用印化伤 |

### 阳刃格（甲见卯、丙戊见午、庚见酉、壬见子）
| 用法 | 取运 |
|---|---|
| 用官 | 喜助官，忌伤食合官 |
| 用煞 | 煞不甚旺则运喜助煞，煞过太重则运喜身旺印绶 |
| 官煞并出 | 运喜制伏，身旺亦可，**财地官乡反为不吉** |

### 建禄月劫格（示例专用）
| 用法 | 取运 |
|---|---|
| 用官 | 印护者喜财，怕官星之逢合，畏七煞之相乘，**伤食为害，劫比即凶** |
| 用财带伤食 | 财食重则喜印绶不忌比肩，财轻则助财不喜印比 |
| 用煞食制 | 食重煞轻则助煞，食轻煞重则助食 |
| **用伤食** | **财运最适，煞亦不忌，行官非吉，透官不美** |
| 官煞并出 | 运喜伤食，比肩亦宜，印绶未为良图，**财官非福运** |

## 四、变格（从格类 · 顺势而为）

### 从旺格
- 日主五行**极旺**（满盘比劫印星）
- 完全没有官杀来克
- 用神：印 + 比劫（顺其旺）
- 忌神：官杀 / 食伤

### 从弱格

| 类型 | 条件 | 用神 | 忌神 |
|---|---|---|---|
| 从财 | 日主极弱 + 满盘财 + 无印无比 | 财 + 食伤 | 印 + 比劫 |
| 从官（杀）| 日主极弱 + 满盘官杀 | 官杀 + 财 | 印 + 比劫 |
| 从儿（食伤）| 日主弱 + 满盘食伤 | 食伤 + 财 | 印 + 官杀 |
| 从势 | 日主弱 + 财官食伤都旺 | 跟最旺势 | — |

## 五、化气格

### 五合化气表

| 合 | 化 | 条件（月令）|
|---|---|---|
| 甲己合 | 化土 | 辰戌丑未 |
| 乙庚合 | 化金 | 申酉 |
| 丙辛合 | 化水 | 亥子 |
| 丁壬合 | 化木 | 寅卯 |
| 戊癸合 | 化火 | 巳午 |

**化气格条件**：
- 日干与邻干合（月干或时干）
- 月令为化神之地
- 化神得力（满盘助化神）
- 不被破坏

⚠️ **化气格极难成立**——多数被判为"合而不化"。

## 七、贵格类

### 拱禄格
- 日时两支**夹日干禄位**
- 如：癸亥日癸丑时 → 亥与丑夹子（癸禄）

### 拱贵格
- 日时两支夹日干天乙贵人

### 夹禄夹贵格
- 类似拱，但邻位

### 三奇贵格
- 天上三奇甲戊庚 / 地下三奇乙丙丁 / 人中三奇壬癸辛
- **顺序排列**才有效

### 魁罡格（日柱）
- 庚辰、庚戌、壬辰、戊戌
- 刚烈、领导力、决断
- ⚠️ 女命魁罡 → 婚姻坎坷

### 金神格（时柱）
- 乙丑、己巳、癸酉
- 刚强、有才

### 六甲趋乾
- 甲日生亥时（亥藏甲禄 + 壬印）

### 六乙鼠贵
- 乙日生子时（子中癸印）

### 六阴朝阳
- 辛日 + 子时（戊子）+ 无官煞印星

### 六壬趋艮
- 壬日 + 寅时（壬寅）

## 八、三命通会独有特殊格局

### 子遥巳格
- 甲子日 + 多见子（不见丑午）
- 原理：子遥合巳中戊土 → 动巳中丙火 → 引官
- 忌：见丑（合子）、见午（冲子）

### 丑遥巳格
- 辛丑、癸丑日 + 多见丑

- **庚子 / 壬子**日多见子 → 冲午引官
- **辛亥 / 癸亥**日多见亥 → 冲巳引官
- 忌见冲字填实

### 倒冲格
- 丙午日多见午冲子

### 井栏叉格
- 庚申、庚子、庚辰日 + 申子辰水局
- 忌填实午字

### 壬骑龙背格
- 壬辰日多见壬辰
- 忌见戌冲辰

### 杂气财官格
- 辰戌丑未月，财官藏其中
- **冲库才发**

### 印重重格
- 印星太旺（**示例命局**）

### 时上一位贵格
- 时柱独透官 / 煞
- 晚发、官星归时

### 月上偏官格
- 月柱独透七煞
- 早年立威

### 岁德格
- 年柱透出日主天德 / 月德

## 九、刑冲合害

### 三合局
- 寅午戌 → 火局
- 巳酉丑 → 金局
- 申子辰 → 水局
- 亥卯未 → 木局

**条件**：三支全到，地支才合化
**半三合**：两支齐到缺一 → 力度减半

### 六合
- 子丑合土、寅亥合木、卯戌合火、辰酉合金、巳申合水、午未合（土）
- 比三合**弱**

### 六冲（最重要）
- 子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- **意义**：变动、激发、冲突
- 月支被冲 → 大动荡
- 日支被冲 → 婚姻 / 健康动

### 三刑
- 寅巳申三刑、丑戌未三刑
- 子卯相刑、辰午酉亥自刑
- **意义**：恩中招怨、内耗

### 六害
- 子未、丑午、寅巳、卯辰、申亥、酉戌
- **意义**：暗中破坏（六爻派认为六害不验，子平派仍用）

## 十、行运成格、行运变格

> 沈孝瞻：命之格局，成于八字，然配之以运，亦有成格变格之要权

### 行运成格
- 原局未成格（用神不透）
- 大运补缺 → 用神出现 → **运合成格**

### 行运变格
- 原局已成格
- 大运冲克 → 格破或变质

## 十一、拘泥格局之失

> 沈孝瞻警告："今人不知轻重，拘泥格局，执假失真。"

- 不要为套格局而误判
- 看**喜忌配合 + 大运推动**，不看格名
- 同一组干支可能有多解 → 看哪个 happens-to 应验

## 十二、示例八字格局综合判

四柱：**癸未 · 甲寅 · 甲戌 · 壬申**

### 是否变格？

- 从旺：❌（有官杀，不能从）
- 从弱：❌（身强）
- 化气：❌（无五合化气）
- 曲直：❌（戌、申破局）
- 拱禄：❌（甲禄已在月支寅）
- 三奇：❌（无完整三奇）
- 魁罡：❌（甲戌不在 4 个中）
- 金神：❌（壬申不在 3 个中）
- 六甲趋乾：❌（时辰非亥）

**结论**：**普通正格** = **建禄格 + 印重重格**（万民英）

### 5 种禄劫用法逐一排查

| 用法 | 是否成立 | 原因 |
|---|---|---|
| 禄劫用官 | ❌ | 辛官藏戌，**孤官无辅** |
| 禄劫用财 | ❌ | 财藏寅戌申未，**比劫直接克** |
| 禄劫用煞 | ❌ | 庚煞藏申，**无食神制** |
| **禄劫用伤食** | ⚠️ 半成 | 寅戌半合火局，**食伤无透** |
| 官煞混 | ❌ | 都藏支无力 |

→ **格局未成 = 待大运补**

### 大运配合（行运成格视角）

- 壬子运（19-28）：印更重 → **印重压制期**
- 辛亥运（29-38）：**辛官透但水更多** → 沈孝瞻"行官非吉" → 不利
- **庚戌运（39-48）**：戌中辛+丁+戊齐到 → **运合成格** ⭐
- 己酉运（49-58）：财官并见，但印重未化
- 戊申运（59-68）：财煞重，印仍盖头
- **丁未运（69-78）**：丁火透干（伤官）+ 财库 → **真正泄秀** ⭐⭐
- 丙午运（79+）：木火通明完整

### 刑冲合害检查

- 寅戌半合火（缺午）→ 火局**藏支半成**
- **寅申冲**月时 → 思维活跃、追逐变化、不安于现状
- 戌未刑（丑戌未三刑缺丑）→ 半刑
- 未戌相刑 → 轻度刑伤

**2026 丙午年 + 阴历 5 月午月 = 三合火局完整**（寅午戌全到）
→ **2026 是火局成的年份** → 食伤全启动、创意大爆发
