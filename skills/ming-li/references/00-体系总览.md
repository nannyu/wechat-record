# 命理学体系总览

> 命理学不是一门学问，是一个**学科群**。"算命"这两个字下面藏着十几个独立的术数体系。
> 本文件是知识地图，让阿瑞在执行命理任务时第一秒就知道**该用哪一术 / 哪一家**。

## 一、五术总分类

| 大类 | 内容 | 解决什么 |
|---|---|---|
| **命** | 八字、紫微斗数、星平会海、铁板神数 | **终生大格局** |
| **卜** | 六爻、奇门遁甲、大六壬、梅花易数 | **单一事件吉凶 / 应期** |
| **相** | 面相、手相、骨相、声相、痣相 | 看人辨人 / 表里相参 |
| **医** | 中医（脉、舌、五运六气）| 健康调理 |
| **山** | 风水（阳宅 / 阴宅）、丹道、符箓 | 空间环境 / 修身养性 |

→ 本 skill 主管 **命 + 卜**（最常用），相 / 医 / 山待启用。

## 四、四术三层定位

## 五、阿瑞执行命理任务的最高心法

1. **不混用方法**——一次推命用 1-3 家方法，标注每家结论
2. **四家互证 + 三盘互证**——一致才下定论
3. **古今差异**——古书断语（如"刑伤""短命"）现代要谨慎解读
4. **诚实承认不确定**——做不到的事直说，不强推
5. **命主专属规则优先**——感情禁区 / 母亲禁区 / 不预测寿命 / 不做价值判断
