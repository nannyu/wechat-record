# 韦千里 8 步范式（现代命书章节标准）

> 民国大师韦千里在《千里命稿》（1935）中定下的标准范式，是**现代命书的鼻祖**。
> 沿用至今近百年，是民国 → 当代 → AI 时代命理输出的**公认结构**。
> ming-li v2.3 起，所有八字推命报告以此 8 步为骨架。

## 一、八步标准章节

| 步 | 章名 | 内容 |
|---|---|---|
| 一 | **看强弱** | 日主旺衰 / 五行强弱 / 月令 / 干支生克累计 |
| 二 | **定格局** | 八格 / 变格 / 建禄 / 月劫等 |
| 三 | **取用神** | 用神选取 + 病神 + 救应 |
| 四 | **论喜忌** | 喜什么 · 忌什么 · 五行调候 |
| 五 | **查岁运** | 大运推排 · 流年应期 |
| 六 | **推六亲** | 父母 / 配偶 / 子女 / 兄弟 / 朋友 |
| 七 | **评性情** | 性格画像 · 心性 · 待人接物 |
| 八 | **断事业** | 职业赛道 · 行业适配 · 长期发展 |

## 二、为什么是 8 步

韦千里的贡献：
1. **白话化革新**——把明清晦涩术语转为通俗语言
2. **流程标准化**——上述 8 步成为后世**通用范式**
3. **案例实证**——百余民国名人命例（蒋介石 / 吴佩孚 / 梅兰芳 / 杜月笙 / 章太炎 / 杨杏佛）

→ 这不是一家之言，是命学界**公认的 SOP**。

## 三、文风纪律（韦千里式）

### ✅ 该用的语言

- **浅近文言**：宜……不宜…… / 其一……其二…… / 何以……？
- **朴素白话**：积累 / 稳重 / 慢热 / 善学 / 独立
- **对话感**：让读者像在场咨询
- **古典印证**：《老子》《周易》苏东坡 王维等经典（一段最多 1 个）

### ❌ 禁用的语言

#### 禁用 1 · 破折号
- ❌ "建禄印重——典型的反思型人格"
- ✅ "建禄印重，乃反思型人格。"

#### 禁用 2 · 英文术语硬塞
- ❌ PMF · Cash Cow · GTD · GTD- · solopreneur · build in public · Cal Newport · deep work · HSP · INTJ · procrastination · Reflective Practitioner · Restless Reflector · Product-Market Fit
- ✅ 改用中文朴素表达：
  - PMF → "积累化为收成的时机"
  - solopreneur → "一人成军 / 独立从业"
  - deep work → "深度工作 / 静心做事"
  - INTJ → "内向思考型"
  - procrastination → "拖延 / 启动慢"

#### 禁用 3 · 商业黑话
- ❌ "产品发布期" / "系统升级期" / "业务起量" / "Cash Cow" / "KPI 定高一些"
- ✅ "出成果的时机" / "充实自己的阶段" / "积累生根" / "宜定大愿"

#### 禁用 4 · 装专业的英文 + 心理学缩写
- ❌ HSP / MBTI / INTJ / OCEAN
- ✅ "高敏感" / "内向思考" / "性情五维"

#### 禁用 5 · 戏剧化煽情
- ❌ "命中注定" / "必定如此" / "三十而立你必须……"
- ✅ "潜力倾向" / "命格显示" / "命主自取"

## 四、韦千里典型批语样式

### 论格局（原样）
> "庚金生于卯月，为正财格；地支亥卯拱木，子辰拱水，食伤生财之局；庚金得命宫之戊土生，戌中之余气，不从。综合而为，正财格逢伤身弱而比劫透。"

→ 文言半白，术语精准，**不堆英文**。

### 取用（原样）
> "以火土为用：天干宜火，火制比劫而无夺财之患；天干宜土，土生金而日元得固。天干忌木，有比劫夺财之患。"

→ 直接说"宜什么 / 忌什么"，朴素。

### 神煞（原样）
> "卯月以申为天德，日主、时支即是。日支子以酉为桃花，位于年、月干；以辰为华盖。日主庚以亥为文昌。"

→ 列举式，不加修辞。

## 五、共情开场（FateTell 教练模式）

韦千里 8 步前面，加一段**共情开场**（现代化补充）：

```
此卷之意，不在定论命主之终生，而在照见命主之结构。
命如一面镜：所映为天生骨格、四时所遇、行运所至。
所见可参考，所行由命主自取。
```

让来访者感到"被看见"，而不是被算计。

## 六、共情结语

韦千里 8 步之后，加一段**共情结语**：

```
命书阅毕，可有所思，可有所惑。
惑者无须急寻答案——命书所予是"参考"，非"定论"。
所行所择，仍由命主自主。
```

## 七、章节排序的内在逻辑

为什么是这个顺序：
1. **看强弱**——先认识自己（五行底色）
2. **定格局**——再看大轮廓（格局）
3. **取用神**——找解题钥匙（用神）
4. **论喜忌**——知道宜避什么（喜忌）
5. **查岁运**——看时间表（运势）
6. **推六亲**——看关系网络（家庭）
7. **评性情**——内在画像（性格）
8. **断事业**——外在落地（职业）

由内而外 · 由本到末 · 由静到动。

## 九、应用时的灵活性

8 步是**骨架**，不是教条。可视情况：
- 来访者只问事业 → **重点第八**，其他从简
- 来访者只问大运 → **重点第五**
- 来访者来咨询家庭 → **重点第六**

但**完整命书必涵 8 步**——这是命学公认的专业标准。
