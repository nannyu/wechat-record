## 二、五大关键差异

### 1. 身旺判断
- **陆致极**：月令 + 五行数量加权（数字）
- **任铁樵**：月令 + **整体气势**（凝聚力）
- → 差异：数字 vs 凝聚力

### 2. 用神选择
- **陆致极**：食伤 / 财 / 官杀分级
- **任铁樵**：用神需**能克制病神**才有效
- → 差异：平衡 vs 制衡

### 3. 旺极
- **陆致极**：简轻讲
- **任铁樵**：**旺极必反凶**
- → 差异：静态 vs 动态

### 4. 调候
- **陆致极**：辅助手段
- **任铁樵**：修正月令判断的**关键**
- → 差异：锦上添花 vs 改判

### 5. 时柱权重
- **陆致极**：与日主一般关系
- **任铁樵**：**命主自主力最强**
- → 差异：平等 vs 优先

## 三、四家任铁樵核心 5 条

### 1. 通关第一
不是"格局好不好"，而是**用神能否疏导日主的力量**

### 2. 气势论
重点不在月令和五行数量，**而在四柱的气聚不聚**

### 3. 旺极反凶
不是宿命，而是**动态平衡**——身旺必须在大运中遇到克制力才能平衡

### 4. 时柱为本
时柱代表命主的自主力，与大运的配合**往往先决定吉凶**

### 5. 配合在调候
简单看"冬天甲木"不够，要看**整个八字的调候是否成立**

## 六、四家互证机制

### 互证等级

| 一致家数 | 确信度 | 操作 |
|---|---|---|
| **4 家全一致** | 极高 | 重大判断可下定论 |
| **3 家一致** | 高 | 主要判断可信 |
| **2 家一致** | 中 | 留意但不强判 |
| **1 家独有** | 低 | 仅作参考 |
| **冲突** | 不下 | 查具体条件，常因某家应用条件错用 |

### 协作流程

```
排盘 → 陆致极（数字简单）
   ↓
身强弱底色 → 任铁樵（气势深度）
   ↓
定格局 + 取运 → 沈孝瞻（系统精密）
   ↓
神煞 + 特殊格局补查 → 万民英（百科全书）
   ↓
四家结论交叉 → 最终判断
```

### 一句话四家合一判

**甲生寅月，建禄而印重，孤官无辅，财食藏支。**
**早岁印盛比旺，难有作为；中年待庚戌煞动财显，方启格局；晚岁丁未透伤入财库，木火通明真展开。**
**时支申截路空亡，晚年事务多阻，重大决策需参谋。**

### 命主操作心法（23 岁现在）

1. **不要硬上**——印重身旺 + 用神埋没，硬冲容易碰壁
2. **积累内功**——这 16 年读书、打基础、建关系、攒资源
3. **等大运补缺**——39 岁庚戌运戌中食伤+财+官齐至，格局成立
4. **晚年慎重决策**——60 岁后做事多请人参谋（截路空亡警示）

## 八、四家方法论的现代意义

### 古今差异
- 古书的"贫贱""短命""刑伤"——古代社会标准，现代要谨慎解读
- 现代职业多元化，"官星 = 公职"已不适用，**官星 = 体制 / 权威 / 规则**
- 现代财富观，"财多身弱"未必凶（投资型人格反而可发）

### 实战建议
- **本命不变 + 大运在变** = 看大运比本命更直接
- **互证为先 + 单家为辅** = 不被一家话框死
- **告诉用户结论 + 解释依据** = 让用户能自己判断
