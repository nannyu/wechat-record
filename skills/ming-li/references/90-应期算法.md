# 流年应期月份算法

> 大运管 10 年，流年管 1 年，**月份才是具体应期**。
> 通用：八字 / 紫微 / 六爻 都用同一套月份应期逻辑。

## 三、应期定位规则

### 1. 三合月份（应吉）

流年地支与四柱某支三合 → 应吉

**例**：流年午 + 命局寅戌 → 寅午戌三合火局 → **5 月（午月）应期最强**

### 2. 六合月份（缓和应吉）

流年地支与四柱某支六合

**例**：流年午 + 命局未 → 午未合 → **6 月**

### 3. 冲月份（应凶/动）

流年地支冲命局某支

**例**：流年午 + 命局子 → 子午冲 → **11 月（子月）动荡**

### 4. 刑月份（暗事）

- 寅巳申 / 丑戌未三刑

### 5. 害月份（暗中破坏）

- 子未、丑午、寅巳、卯辰、申亥、酉戌

## 四、八字流年应期

### 流年与命局互动

| 互动 | 主 |
|---|---|
| 流年干生日主 | 较吉 |
| 流年干克日主 | 较凶 |
| 流年支冲月令 | 大变化 |
| 流年支合用神 | 较吉 |
| 流年支冲日支 | 人际/感情波动 |

**最凶组合**：流年干克日主 + 流年支冲月令
**最吉组合**：流年支合用神 + 流年干生日主

## 五、紫微流年应期

- 流年地支落宫 = **流年命宫**
- **流年四化**（按流年干）冲动本命星 = 应期
- **三盘叠加**：本命 + 大限 + 流年三层都看

## 六、六爻应期

| 用神状态 | 应期 |
|---|---|
| 用神旺动 | 当日 / 当时 |
| 用神动而逢合 | 冲开之日 / 月 |
| 用神静 | 冲动之日 |
| 用神空 | 出旬之日 |
| 用神入墓 | 冲墓之日 |
| 三合局缺一 | 补字之日 / 月 |
| 用神临破 | 填实之日 |

**问近事**：当日 / 旬内
**问远事**：按月或按年

## 七、万民英应期六法

> 三命通会独有

1. **冲应**：流年支冲命局某支 → 当年/月动
2. **合应**：流年支合命局某支 → 当年/月成
3. **刑应**：流年支刑命局某支 → 当年/月伤
4. **会应**：流年支与命局形成三会方 → 大动
5. **拱应**：流年支与某支拱出贵人 → 突贵
6. **填实**：流年支补足三合局缺字 → **格局成立年**

### 大运转换期

**交运前后 1 年最关键**：
- 大运换柱年 ± 1 年内
- 应避免重大决策
- 易有调整 / 适应期

## 八、示例应期实例

四柱：**癸未 · 甲寅 · 甲戌 · 壬申**

### 寅午戌三合火局

- 命局**已有寅戌**（缺午）
- 流年逢午 = 寅午戌三合火完整 → **食伤局完成**
- **2026 丙午年**：午年 + 5 月午月 = 三合火局极致 ⭐

### 寅申冲

- 月支寅 + 时支申 = 寅申冲（原局已存在）
- 流年逢寅或申 → **冲动 + 加重**

### 大运交运年

- 29 岁交辛亥（2031-2032）⚠️ 不利运
- 39 岁交庚戌（2041-2042）⭐ 格局启动
- 69 岁交丁未（2071-2072）⭐⭐ 大成期

### 库开应期

- 戌库（火库 + 财库 + 官库）：辰运/年冲开 → **得财 + 升迁**
- 未库（财库 + 伤官库）：丑运/年冲开 → **得意外之财**

## 九、应期判断三层叠加

**最高确信应期** = 三层都指向同一月份：
1. 大运配合（10 年主题）
2. 流年配合（这一年的主题）
3. 流月配合（具体月份）

三层都指向某月 → **重大事件应期** ⭐⭐⭐

**例**：示例命主 2026 丙午年
- 大运壬子（印重）→ 主背景
- 流年丙午 → 食伤+财年（吉）
- 5 月午月 → 三合火局完整
- **综合**：2026 年 5 月 = 食伤全启动、创意大爆发（已有共识）

## 十、应期注意事项

1. **应期不是绝对**——天时（流年）+ 地利（环境）+ 人和（自己努力）三合
2. **吉应期可主动**——上贵气月份做重大决策
3. **凶应期宜守**——动荡月份避免硬上
4. **应期 ≠ 宿命**——告诉用户"倾向"而非"必定"
