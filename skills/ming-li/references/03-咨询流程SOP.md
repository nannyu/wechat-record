# 咨询流程 SOP（接案 → 推演 → 结案）

> 借鉴 ISAR 咨询占星 + 心理咨询 SOAP 流程。
> 命理推演不是"一锤子买卖"，而是有结构的咨询过程。

## 一、流程总览（6 步）

```
1. 接案（Intake）
   ↓
2. 收集生辰（Data Collection）
   ↓
3. 排盘 + 推演（Analysis）
   ↓
4. 输出报告（Report）
   ↓
5. 咨询答疑（Q&A）
   ↓
6. 结案 + 存档（Closure）
```

## 二、Step 1 · 接案

### 1.1 来访者意图分类

| 意图 | 推荐方法 |
|---|---|
| "我是个什么样的人？" | 八字命格 + 紫微命宫 |
| "我事业 / 财运 / 婚姻怎么样？" | 紫微对应宫 + 八字相关用神 |
| "这件事会不会成？" | 六爻（一事一占）|
| "今年怎么样？" | 八字流年 + 紫微流年 |
| "我应该做 X 还是 Y？" | 六爻 / 综合判 |
| "我什么时候能 X？" | 应期算法（references/90）|

### 1.2 接案话术（避免 dramatize）

✅ "我们一起看看你的命理结构，找到你的优势、议题、时机"
❌ "你想知道命中注定的什么？"

✅ "命理是工具，最终决策权在你"
❌ "天意如此，你必须……"

## 三、Step 2 · 收集生辰

**必问 5 项**：

1. 公历 or 农历？（紫微以农历为基础，八字以节气为基础）
2. 年月日 + 时辰（精确到分）
3. 出生地（**真太阳时校正必需**）
4. 性别（决定大运 / 大限顺逆）
5. 当前年龄 + 主要关注问题（聚焦输出）

**没给齐**：主动追问，**不要瞎猜**。

## 四、Step 3 · 排盘 + 推演

### 3.1 推荐工具排盘（不手算）

| 体系 | 推荐 |
|---|---|
| 八字 | [神机阁](https://www.shenjige.cn/sizhu/base) |
| 紫微 | [问真紫微](https://pcbz.iwzwh.com/ziwei/) |
| 六爻 | 用户自摇 / Python 模拟 |

### 3.2 四家合一推演（八字）

按 `10-八字推演.md` 8 步法：
1. 真太阳时校正
2. 排四柱
3. 地支藏遁
4. 身强身弱（陆致极）
5. 气势论补判（任铁樵）
6. 取用神（沈孝瞻 / 万民英）
7. 神煞查（11 神煞库）
8. 大运 + 流年应期（90）

### 3.3 互证机制

| 一致家数 | 确信度 |
|---|---|
| 4 家全一致 | 极高 |
| 3 家一致 | 高 |
| 2 家一致 | 中 |
| 1 家独有 | 低 |
| 冲突 | 不下定论 |

## 五、Step 4 · 输出报告

### 默认 Markdown 报告

按 ISAR 8 章结构（详见 `02-命理学专业化框架.md`）：

```markdown
# [姓名] 命理综合分析

## 1. 命造卡片
## 2. 性格画像（你是什么样的人）
## 3. 优势与才能
## 4. 生命议题
## 5. 成长模式
## 6. 当前阶段意义
## 7. 危机与机会时期
## 8. 时机与决策建议
## 9. 综合行动建议
```

### HTML 报告（用户要求时）

走 `95-HTML报告模板.md` 完整 9 章可视化版本。

## 六、Step 5 · 咨询答疑

### 高频问题准备

| 问题类型 | 应答策略 |
|---|---|
| "我会不会发财？" | 看财格 + 财运大运 → 给"什么时段 + 什么方式"|
| "我什么时候结婚？" | 看夫妻宫 + 桃花应期 → 给"概率窗口"非"必定年份" |
| "我适合做什么？" | 看官 / 财 / 食伤 → 给"职业方向"|
| "我健康怎样？" | **谨慎**——只说"宜注意 XX 系统"，不诊断病 |
| "我能不能跟 XX 在一起？" | ⚠️ **不算姻缘配对** |

### 来访者反驳"我不信"

✅ "命理是一种自我认识工具，信不信你来决定，我们只看结构和倾向"
❌ "你不信也得信，命中注定如此"

## 七、Step 6 · 结案 + 存档

### 结案话术

✅ "我们今天聊到的，建议你回去想想哪些是已经认识到的、哪些是新视角"
✅ "命理结构相对稳定，但你的选择会改变结构的表达方式"
✅ "如果某个判断未来证实或证伪了，欢迎反馈，我们会更新档案"

### 存档（命主专属场景）

- 推命存档至 `Vault 50.个人/命例-XXXX-MM-DD.md`
- 重要发现回写 `Vault 20.领域/命理学/个人档案-四家合一.md（自行维护）`

### 存档（通用用户）

- 不存档（隐私 / 一次性咨询）
- 仅在用户明确要求 + 脱敏后做命例库（v3.0）

## 八、咨询时长建议

| 类型 | 时长 | 输出 |
|---|---|---|
| 快速咨询 | 10-15 分钟 | 简要 Markdown 报告 |
| 标准咨询 | 30-45 分钟 | 完整 Markdown 报告（ISAR 8 章）|
| 深度咨询 | 60-90 分钟 | **HTML 报告 + 现场答疑** |
| 长期咨询 | 多次 | 每年大运 / 流年复盘 |

## 九、不接的咨询

明确拒绝：
1. ❌ 姻缘配对（"我跟 XX 能在一起吗？"）
2. ❌ 寿命预测（"我能活到几岁？"）
3. ❌ 死期 / 重大灾祸预测
4. ❌ 替他人推命（除非授权）
5. ❌ 反复占同一事（六爻"渎卦"）
