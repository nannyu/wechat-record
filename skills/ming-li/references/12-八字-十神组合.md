# 十神组合解读（20+ 经典组合）

> 八字十神 5 对：印（正/偏）、比劫、食伤、财（正/偏）、官杀（正官/七杀）
> 单看一个神不够，**组合才有判断**。

## 一、印星组合

### 杀印相生（极佳）
- 七杀 + 偏印 同透/同位
- **作用**：杀生印、印化杀、压力转动力
- **意义**：能在压力下成长、有担当
- **应用**：领导力、危机管理、跨越式发展

### 财印不并位（好）
- 财与印分开（年柱财、月柱印 OK）
- **意义**：财不破印，学问与金钱不冲突

### 财印交战（凶）
- 财与印同柱或近邻
- 财坏印
- **意义**：要么贪财失学问，要么学问拖累财
- **化解**：用比劫通关（劫生财、护印）

### 印重身旺（懒）
- 印星太多 + 比劫旺
- **意义**：才华横溢但行动慢、依赖性强、好学难成
- **应用**：示例命主当前壬子大运正是此状态

### 印轻身弱（弱）
- 印少 + 身弱
- **意义**：底气不足、需贵人扶持

## 二、比劫组合

### 比劫争财（凶）
- 比劫多 + 财弱
- **意义**：兄弟朋友分财、合伙难、被借钱
- **化解**：用食伤（食生财）

### 比肩夺财（凶）
- 同上但更狠
- **化解**：用官杀制比

### 群劫无依
- 比劫极多
- **意义**：可能从旺格（特殊）

## 三、食伤组合

### 食神制杀（极佳）
- 食神克七杀
- **意义**：以柔克刚、用智慧解决威胁
- **应用**：技术能力、创新能力、转化压力

### 伤官生财（佳）
- 伤官生正财/偏财
- **意义**：用才华换钱、技术变现
- **应用**：自由职业、创意工作、知识付费

### 伤官见官（凶 · 经典禁忌）
- 伤官 + 正官同现
- **意义**：与领导冲突、官非、规则崩塌
- **化解**：伤官配印（印克伤官）
- ⚠️ **伤官见官，祸患百端**

### 食伤泄秀（好）
- 身强用食伤
- **意义**：才华外露、表达欲强、艺术天赋

### 枭神夺食（凶）
- 偏印克食神
- **意义**：好点子被压、健康受损、孤独
- **化解**：用财制偏印

## 四、财星组合

### 财官双美（极佳）
- 正财 + 正官 同透
- **意义**：富贵双全、事业 + 财富同步
- **应用**：从政、企业高管

### 财生官（好）
- 财在前生官
- **意义**：用钱换权、商业转政治

### 财多身弱（凶）
- 财太多 + 日主弱
- **意义**：富屋穷人、有财守不住、压力大
- **化解**：用比劫帮身

### 财库冲开（吉）
- 辰/戌/丑/未 财库被冲
- **意义**：财从库中出、得意外之财
- **应用**：大运/流年冲库年是发财年

### 财坏印（凶）
- 财克印
- **意义**：金钱观念扭曲学问 / 与母不合

## 五、官杀组合

### 官杀混杂（凶）
- 正官 + 七杀同现
- **意义**：领导多、规则乱、压力混乱
- **化解**：合杀留官 或 合官留杀

### 七杀有制（好）
- 七杀被食神制
- **意义**：化压力为成就、有威权

### 七杀无制（凶）
- 七杀 + 无食神
- **意义**：暴躁、横祸、官非

### 杀重身轻（凶）
- 七杀太多 + 日主弱
- **意义**：被压制、身心俱疲
- **化解**：用印化杀

## 六、特殊组合

### 天乙加贵
- 天乙贵人 + 正官 同位
- **意义**：得贵人扶持任要职

### 三奇贵格
- 甲戊庚 / 乙丙丁 / 壬癸辛 顺序排
- **意义**：才华横溢、非凡之命

## 七、示例八字十神组合验证

四柱：**癸未 · 甲寅 · 甲戌 · 壬申** · 日主甲木

**十神分布**：
- 年干癸 = **正印**
- 月干甲 = **比肩**
- 时干壬 = **偏印**
- 年支未藏：乙劫财、己正财、丁伤官
- 月支寅藏：甲比肩、丙食神、戊偏财
- 日支戌藏：戊偏财、辛正官、丁伤官
- 时支申藏：庚七杀、壬偏印、戊偏财

**重要组合**：

1. **双印透干（癸 + 壬）**：印星太重
2. **比肩坐月令**：建禄格
3. **七杀被偏印化**：七杀庚 + 偏印壬 → **杀印相生**（极佳潜质）
4. **正官藏库**：戌中辛金正官 → 隐性官星，需大运引出
5. **伤官生财（藏支）**：戌中丁火生戊土、未中丁生己 → 伤官生财格局**藏在地支**
6. **印重身旺**：印 + 比劫 + 月令禄 → **典型印重身旺**
7. **官杀混杂藏支**：戌中辛正官 + 申中庚七杀 → 不混乱
8. **天乙贵人 + 印**：年支未天乙 + 年干癸印 → 早年得长辈贵人

**综合评判**：
- 主格：**建禄 + 印重身旺**
- 辅格：**杀印相生**（极佳潜质，待大运启动）
- 用神：**食伤生财**（火土，藏支不透）
- 忌神：**印星太过**（壬癸水）
- 当前壬子运：**印更旺** → 懒散期、积累期
- 庚戌运起：**七杀透出 + 食伤库开** → 杀印相生 / 伤官生财 正式启动
