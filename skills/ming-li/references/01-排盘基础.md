# 排盘基础（八字 / 紫微通用）

> 所有命理推演的第一步。真太阳时 → 干支换算 → 阴阳排列 → 四柱完成。
> 本文件是**所有方法的共用底层**。

## 一、真太阳时校正

### 公式

```
真太阳时 = 北京时间 + (出生地经度 − 120) × 4 分钟 + 均时差
```

### 常用经度参考

| 地点 | 经度 | 修正分钟 |
|---|---|---|
| 北京 | 116.4°E | −14 分钟 |
| 上海 | 121.5°E | +6 分钟 |
| 某地 | 113.1°E | −28 分钟 |
| **某地** | 112.88°E | **−28 分钟**（示例命主）|
| 广州 | 113.3°E | −27 分钟 |
| 哈尔滨 | 126.5°E | +26 分钟 |
| 西安 | 108.9°E | −44 分钟 |
| 成都 | 104.1°E | −64 分钟 |
| 乌鲁木齐 | 87.6°E | −130 分钟 |

### 均时差（每月 ±15 分钟内）

精确算需查表，常用月份大致：
- 1 月：约 −10 分钟
- 2 月：约 −13 分钟
- 5 月：约 +3 分钟
- 11 月：约 +16 分钟
- 其他月份按月平均 0 ± 5 分钟可近似

### 何时必算

- **时辰临界**（如 14:55 离申时 15:00 太近）
- **跨节气一两天内**出生
- **海外出生**
- **西部出生**（修正幅度大）

### 示例命主

- 出生：<某年 某月 某日 某时辰（北京时间） 某地>（示例）
- 修正：16:15 − 28 分钟 = **15:47**
- 时辰：**申时**（15:00 - 17:00）→ 不变

## 二、干支转换

### 天干（10 个）

甲乙丙丁戊己庚辛壬癸
- 阳干：甲丙戊庚壬
- 阴干：乙丁己辛癸

### 阴历转换

紫微以**农历**为基础，八字以**节气**为基础——**两套体系不同**。

**节气界限**（八字用）：
- 年柱：以**立春**为界（不是春节）
- 月柱：以**节气**为界（立春、惊蛰、清明...）
- 日柱：从万年历查
- 时柱：按**五鼠遁**

## 三、四柱排法（八字）

### 1. 年柱

以**立春**为界：
- 立春前 = 上一年
- 立春后 = 当年

### 2. 月柱（五虎遁）

按**节气**起月，**五虎遁口诀**起月干：

| 年干 | 正月起 |
|---|---|
| 甲己 | 丙寅 |
| 乙庚 | 戊寅 |
| 丙辛 | 庚寅 |
| 丁壬 | 壬寅 |
| 戊癸 | 甲寅 |

### 3. 日柱

从**万年历**查（推算公式复杂，工具排盘最稳）。

### 4. 时柱（五鼠遁）

按日干起子时干：

| 日干 | 子时干 |
|---|---|
| 甲己 | 甲子 |
| 乙庚 | 丙子 |
| 丙辛 | 戊子 |
| 丁壬 | 庚子 |
| 戊癸 | 壬子 |

**申时映射**（本命主用）：
- 甲己日 → 壬申时
- 乙庚日 → 甲申时
- 丙辛日 → 丙申时
- 丁壬日 → 戊申时
- **戊癸日 → 庚申时**

## 四、五行生克

### 相生
木→火→土→金→水→木

### 相克
木→土→水→火→金→木

### 制化
- 我生：泄气（如金生水，水为金之泄）
- 我克：耗气（如金克木，木为金之财）
- 生我：源助（如金被土生）
- 克我：制约（如金被火克）
- 比和：助力（如金见金）

## 五、十神定义（八字用）

以**日干**为我：

| 关系 | 阳同 | 阴异 |
|---|---|---|
| 同我（比劫）| 比肩 | 劫财 |
| 我生（食伤）| 食神 | 伤官 |
| 我克（财）| 偏财 | 正财 |
| 克我（官煞）| 七煞 | 正官 |
| 生我（印枭）| 偏印 | 正印 |

**阴阳判定**：日干阳，遇同性 = 比肩 / 阳；日干阴，遇同性 = 比肩 / 阴。
**阴阳相反 = 正**（如阳遇阴 = 正官）
**阴阳相同 = 偏**（如阳遇阳 = 七煞）
