# 六爻文王课推演

> 六爻 = **一事一占**。野鹤老人《增删卜易》方法论：用神中心法 + 月日辰衰旺 + 应期推算。
> 与八字 / 紫微的区别：**当下心念 + 摇卦 → 单一事件的吉凶 + 应期**。

## 一、触发条件

- 用户说"摇卦 / 卜卦 / 起卦 / 六爻"
- "这件事 [会不会 / 能不能 / 该不该]"
- "什么时候能 [成 / 办 / 到]"
- "问个事"

## 二、六爻 vs 其他易卜

| 体系 | 起卦法 | 核心 | 适用 |
|---|---|---|---|
| **六爻（文王课）** | 三枚铜钱摇 6 次 | 用神 + 五行生克 + 月建日辰 | **具体一事问吉凶** |
| 梅花易数 | 时间 / 数字 / 万物 | 体用 + 五行 + 卦象 | 即兴起卦速断 |
| 周易卦辞 | 蓍草 / 时间 | 卦辞爻辞 | 抽象义理推演 |
| 奇门遁甲 | 时间盘 | 三奇六仪 + 九宫 | 谋事 / 用兵 / 趋避 |
| 大六壬 | 时辰 | 四课三传 | 国事大事 |

## 三、起卦法（铜钱三文）

**乾隆通宝最佳**（材质足、震动好），无则用任意硬币。

- 三个铜钱平入手心 → 摇 6 次 → 由下往上画爻
- 一背两字 = **单**（少阳，画 `―`）
- 两背一字 = **拆**（少阴，画 `--`）
- 三背无字 = **重**（老阳，变爻，画 `O`，动则变阴）
- 三字无背 = **交**（老阴，变爻，画 `X`，动则变阳）

**记录顺序**：初爻（最下）→ 二爻 → 三爻（内卦完）→ 四爻 → 五爻 → 六爻（外卦完）

## 五、用神中心法（六爻精髓）

### 用神四神

| 名称 | 定义 | 作用 |
|---|---|---|
| **用神** | 卦中所占之事的代表爻 | 全卦判断中心 |
| **元神** | 生用神的爻 | 用神之援助 |
| **忌神** | 克用神的爻 | 用神之敌 |
| **仇神** | 克元神 + 生忌神的爻 | 间接为害 |

### 用神取法

| 占什么 | 用神 |
|---|---|
| 父母、长辈、师长、文书、车舟、房屋 | **父母爻** |
| 子女、晚辈、医药、僧道、六畜、福德 | **子孙爻** |
| 妻、财物、买卖、金钱、器皿 | **妻财爻** |
| 功名、官府、丈夫（女占）、上司、贼盗 | **官鬼爻** |
| 兄弟姐妹、朋友、合伙 | **兄弟爻** |
| 自占吉凶 / 防灾 | **世爻** |
| 占他人 | **应爻** |

⚠️ **用神错 = 百无一验**。

### 用神有力 / 无力

**有力**（占吉）：
1. 用神旺相（月令 / 日辰生扶）
2. 动而化吉（化回头生 / 化进神）
3. 长生 / 帝旺于日辰
4. 与元神同动
5. 临空 / 化空但**待出空可用**

**无力**（占凶）：
1. 月破 / 日克
2. 动而化退 / 化绝 / 化破
3. **入墓**（最凶）：金墓丑、木墓未、水土墓辰、火墓戌
4. 逢空且不动
5. 被忌神动克 + 无元神救

## 六、衰旺判断（断卦第一关）

### 1. 月建（最大权）
- 月生用 → 大吉
- 月克用 → 大凶
- 月冲用 → **月破**（大凶）
- 月临用 → **极旺**

### 2. 日辰（次大权）
- 日生用 → 吉
- 日冲用：静爻冲为**暗动**（旺爻吉/衰爻凶），动爻冲为**冲散**
- 日克用 → 凶
- 日合用 → 缓

### 3. 动静
- 静爻 = 守势
- 动爻 = 变化主源
- 静爻被冲 → **暗动**
- 动爻被合 → 合住（待冲开应）

### 4. 变爻反应
- 化回头生 → 大吉
- 化回头克 → 大凶
- 化进神 → 力增
- 化退神 → 力减
- 化绝/化墓/化破/化空 → 凶

### 5. 空亡
- 旬空 = 用神逢空
- 静空 + 不动 → **真空（凶）**
- 动空 / 月建临空 → **假空（可用）**

## 七、应期推算

| 用神状态 | 应期 |
|---|---|
| 用神旺动 | 当日 / 当时 |
| 用神动而逢合 | 冲开之日 |
| 用神静 | 冲动之日 |
| 用神空 | 出旬之日 |
| 用神入墓 | 冲墓之日 |
| 三合局缺一 | 补字之日 |
| 用神临破 | 填实之日 |

**问近事**（近病/求财）：当日 / 旬内
**问远事**（功名/婚姻）：按月或按年

## 八、生克冲合刑害

### 三合局
- 寅午戌火局 / 巳酉丑金局 / 申子辰水局 / 亥卯未木局

### 六合
- 子丑、寅亥、卯戌、辰酉、巳申、午未
- 合住 → **暂留不去**（吉事缓发，凶事缓散）

### 六冲（最重要）
- 子午、丑未、寅申、卯酉、辰戌、巳亥
- 冲散 → **吉凶皆速决**
- **六冲卦**：万事无成或速决
- **六合卦**：万事亨通或缠绵

**经典组合**：
- 六冲变六合 → 先散后聚
- 六合变六冲 → 先聚后散

### 三刑
- 寅巳申 / 丑戌未三刑
- 子卯相刑、辰午酉亥自刑

### 六害
- 子未、丑午、寅巳、卯辰、申亥、酉戌
- 野鹤："六害全无应验，删之不录"——基本可忽略

## 九、断卦 7 步 SOP

1. **起卦**：记录年月日 + 旬空 + 所占
2. **装卦**：按卦象 → 本宫 → 六亲 + 世应 + 纳甲 + 六神
3. **取用神**：按"占什么用什么爻"表
4. **定衰旺**：月建 / 日辰 / 旬空 / 元神忌神
5. **看动变**：动爻变出之爻生克冲合用神
6. **判吉凶**：简化公式
7. **定应期**：按用神状态选

### 简化吉凶公式

| 局面 | 结论 |
|---|---|
| 用神旺 + 元神动生 + 无忌神 | **大吉** |
| 用神旺 + 忌神不动 + 元神有 | **吉** |
| 用神旺 + 元神空 + 忌神动 | 中平偏不利 |
| 用神休囚 + 忌神动 + 无救 | **凶** |
| 用神入墓 / 化绝 / 月破日克 + 无救 | **大凶** |

## 十、野鹤老人禁忌

### ❌ 不该做的

1. **不可再三渎卦**（一事一日内反复占 → 神不告）
2. **不为不孝不悌之事占**（求父母短寿、兄弟祸患）
3. **不替代医生**（占病 = 提示防患）
4. **不替代法律**（占官非 = 看趋避）
5. **不用迷信化语言**："必定" → "卦象显示"
6. **不算姻缘配对**（user.md 禁区）
7. **不预测寿命**

### ✅ 该做的

1. 起卦前**所占之事清晰**
2. **诚意起卦**——玩心不灵
3. 一事一占，重大可分占
4. **建议交叉验证**：六爻 + 八字流年互对照
