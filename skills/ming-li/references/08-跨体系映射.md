# 跨体系映射（八字 ↔ 紫微 ↔ 西洋占星 ↔ 塔罗 ↔ MBTI ↔ 大五人格）

> 为什么要做这个：
> - 年轻人（20-35 岁，82% 本科以上）更熟悉星座 / MBTI / 塔罗，对八字陌生
> - 单一命理体系只能给一种视角；**多体系互译**能让来访者从已熟悉的概念进入
> - 命理咨询不应是封闭术语堆——应该是**多语言对话**
>
> **核心原则**：
> 1. 是功能映射，不是严格等价（基础理论各不同）
> 2. 用熟悉概念做"翻译桥"——帮来访者理解陌生术语
> 3. 不冒充其他体系的专家——西洋占星 / 塔罗只做辅助提示
> 4. 始终回到八字 / 紫微做主判，跨体系做辅助叙述

## 九、跨体系推命的纪律

### ✅ 该做的

1. **以八字 / 紫微为主判**——跨体系做辅助叙述
2. **用来访者熟悉的语言开场**——星座 / MBTI / 塔罗
3. **指出体系局限**——星座只看太阳月亮，八字给你 4 柱 + 大运
4. **多体系互证 → 高确信判断**

### ❌ 不该做的

1. **不冒充占星师 / 塔罗师**——只用基础映射，不做深度占星 / 塔罗解读
2. **不替代任一体系的专业**——西洋占星 / 塔罗有专门的认证师
3. **不做硬性对应**——映射是"功能近似"，不是"严格等价"
4. **不为讨好年轻人而牺牲严谨**——八字主判依然要严格
