### 视觉硬规范（v2.4 固化）

**字体**：Kaiti SC / STKaiti / Songti SC。禁用任何拉丁衬线（Cormorant Garamond 等）和英文标题。

**配色**：
- 宣纸米黄 `#ede4cd`
- 墨主 `#1f1b16` / 墨次 `#3b342a` / 墨淡 `#8a8170 #a39884`
- 朱砂 `#8e2e22`
- 古铜 `#8a6f47`

**线条规范**：
- 横线一律渐变（两端淡入淡出），不用 `border 1px solid`
- 总诀匾额四角角花（L 形），不用横框
- 子目用朱砂菱形 + 渐变引线，不用左侧竖线
- 通俗说卡用朱砂印章式标签 + 浅色块，不用左侧竖线

**章节结构**：韦千里 8 步 + 卷首共情开场 + 卷末术语简注 + 共情结语 + 印章落款

**通俗化双层（不三层）**：
- 行内 `.gloss` 浅注（关键术语第一次出现时）
- 卷末术语简注（30+ 名词速查）
- ⚠️ 不要正文+通俗说卡+简注三层重复

**月份单字**：正二三四五六七八九十冬腊（不用壹贰叁..."十一""十二"会撑破格子）

### PNG 导出能力

实测示例八字报告：
- HTML 输入：v2.4 完整版（~33KB）
- PNG 输出：820 × 10488 px，2.7 MB
- 用时：< 5 秒（Chrome headless）
- 用途：发微信群 / 朋友圈 / 长期收藏

### 完整咨询输出（v2.4 起）

默认双输出，**分别落在不同位置**：

```
HTML → Vault 20.领域/命理学/命理报告-{{姓名}}-{{日期}}.html  (精读 / 归档)
PNG  → ~/Desktop/命理报告-{{姓名}}-{{日期}}.png             (发图 / 分享)
```

PNG 落桌面是用户明确要求，方便随手发群、朋友圈、AirDrop。

生成后自动 `open` 浏览器预览。

3. **专业咨询 = 教练模式**（FateTell 启示）：先共情，再分析
4. **古典美学现代化 = 抽象提炼**（故宫文创范式）：不照搬古风
5. **高端命理 = 重内容轻形式**：Word/PDF 朴素输出，无炫技图表
6. **紫云派专业输出 = 模块化分项专论**
